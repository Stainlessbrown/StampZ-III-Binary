# StampZ-III

A desktop application for philatelic (postage stamp) image analysis. Provides colour measurement, spectral analysis, perforation gauging, 3D colour-space plotting, and database management.

## ⬇️ Download

**[→ Download the latest release here](https://github.com/Stainlessbrown/StampZ-III-Binary/releases/latest)**

No GitHub account required — click the link above and download the file for your platform.

---

## Installation

### Windows
1. Download `StampZ-III-*-Setup.exe`
2. Run the installer
3. If Windows SmartScreen appears, click **"More info"** → **"Run anyway"**
4. Launch from the Start Menu or Desktop shortcut

> **Note:** If the downloaded file has no `.exe` extension, rename it and add `.exe` before running.

### macOS (Apple Silicon — macOS 14+)
1. Download `StampZ-macOS-Universal.zip`
2. Extract the zip
3. Right-click `StampZ-III.app` → **Open** (required the first time to bypass Gatekeeper)

### Linux (Ubuntu 22.04+ / Linux Mint 21+)
1. Download `StampZ-Linux-x64.tar.gz`
2. Extract: `tar -xzf StampZ-Linux-x64.tar.gz`
3. Run: `./StampZ-III/StampZ-III`
